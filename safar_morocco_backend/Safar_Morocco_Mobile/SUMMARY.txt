🚀 DÉMARRAGE RAPIDE
══════════════════════════════════════════════════════════════════════════════

Installation (2 min):
  $ git clone <url>
  $ cd Safar_Morocco
  $ mvn clean install

Configuration (1 min):
  $ cp .env.example .env
  # Edit .env with your database credentials

Démarrage (1 min):
  $ mvn spring-boot:run
  # App runs on http://localhost:9004

Test (1 min):
  $ curl http://localhost:9004/actuator/health
  # Response: {"status":"UP"}

Total: 5 minutes! ⏱️
